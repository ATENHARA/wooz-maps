# Contributing

**Issues are welcome** — if you spot a wrong layout, a hard-to-read color, missing NPC data, or anything else off, please [open an issue](https://github.com/ATENHARA/wooz-maps/issues/new). Screenshots and in-game coordinates help a lot.

**Pull requests are not accepted.** This repo isn't set up for reviewing or merging outside changes — please don't submit a PR. Report the problem as an issue instead and it'll get addressed directly.

The pack is public domain (see [LICENSE](LICENSE)), so you're always free to fork it and make your own changes for personal use.
